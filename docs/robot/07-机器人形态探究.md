## 1. 形态设计核心矛盾

```mermaid

    A["功能全面性<br/>多种清洁能力"] <--> B["体积紧凑性<br/>通过 0.6m 门宽"]
    C["作业效率<br/>大水箱/大刷盘"] <--> D["机动灵活<br/>隔间内转向"]
    E["续航能力<br/>大电池"] <--> F["重量控制<br/>≤ 80kg"]
    
    style A fill:#e3f2fd,stroke:#1565c0
    style B fill:#fff3e0,stroke:#e65100
    style C fill:#e3f2fd,stroke:#1565c0
    style D fill:#fff3e0,stroke:#e65100
    style E fill:#e3f2fd,stroke:#1565c0
    style F fill:#fff3e0,stroke:#e65100
```

**核心矛盾**：公共卫生间清洁需要的功能多、对象杂，但空间极度受限。这是 PRISM 区别于所有现有商用清洁机器人的最大设计挑战。

### 2.3 构型决策

```mermaid
flowchart TB
    DEC{"构型决策"}
    
    DEC -->|"推荐方案"| REC["方案 B：底盘 + 机械臂复合型"]
    DEC -->|"远期探索"| FUT["方案 C：母子机协同型"]
    DEC -->|"暂不考虑"| NO1["方案 D：改造成本过高"]

    REC --> R1["底盘：负责移动 + 地面清洁（拖洗）"]
    REC --> R2["机械臂：负责便器/台面清洁（刷/擦/喷）"]
    REC --> R3["优势：技术可行性高，适应多种清洁对象"]

    style REC fill:#2e7d32,color:#fff
    style FUT fill:#1565c0,color:#fff
    style NO1 fill:#9e9e9e,color:#fff
```

**决策理由**：
1. 方案 B 兼具地面清洁（底盘）和立体清洁（机械臂），功能覆盖最全
2. 底盘技术可复用成熟的商用清洁机器人方案，降低研发风险
3. 机械臂可分阶段开发 —— 首版实现地面清洁，二版加入机械臂便器清洁
4. 方案 C（母子机）作为远期优化方向保留，解决更极端的空间约束

## 3. 底盘方案设计

### 3.1 运动方案对比

```mermaid
flowchart TB
    subgraph Diff["差速驱动"]
        D1["两个独立驱动轮<br/>+ 万向从动轮"]
        D2["✅ 原地旋转<br/>✅ 结构简单<br/>❌ 不能横移"]
    end

    subgraph Omni["全向轮 (Mecanum/Omni)"]
        O1["四个麦克纳姆轮<br/>或全向轮"]
        O2["✅ 任意方向移动<br/>✅ 狭窄空间灵活<br/>❌ 湿滑地面打滑<br/>❌ 承载能力低"]
    end

    subgraph Ack["阿克曼转向"]
        AK1["前轮转向<br/>后轮驱动"]
        AK2["✅ 高速稳定<br/>❌ 转弯半径大<br/>❌ 无法原地转"]
    end

    subgraph Track["履带式"]
        TR1["双侧履带驱动"]
        TR2["✅ 越障强<br/>✅ 防滑好<br/>❌ 转向磨损地面<br/>❌ 不适合室内"]
    end

    style Diff fill:#e8f5e9,stroke:#2e7d32
    style Omni fill:#e3f2fd,stroke:#1565c0
    style Ack fill:#f3e5f5,stroke:#6a1b9a
    style Track fill:#fce4ec,stroke:#b71c1c
```

### 3.2 底盘方案决策

| 维度 | 差速驱动 | 全向轮 | 阿克曼 | 履带 |
|------|:---:|:---:|:---:|:---:|
| 隔间转向 | ★★★★ | ★★★★★ | ★★ | ★★★ |
| 湿滑地面 | ★★★★ | ★★ | ★★★★ | ★★★★★ |
| 结构简单 | ★★★★★ | ★★★ | ★★★ | ★★★ |
| 承载能力 | ★★★★ | ★★★ | ★★★★★ | ★★★★★ |
| 成本 | ★★★★★ | ★★★ | ★★★★ | ★★★ |
| 地面友好 | ★★★★ | ★★★★ | ★★★★★ | ★★ |

**决策**：**差速驱动 + 防滑橡胶轮**

- 结构简单可靠，成本低
- 原地旋转满足隔间内调头需求
- 配合防滑纹路橡胶轮应对湿滑地面
- 前部增加一对万向从动轮保证稳定性（四轮触地）

### 3.3 底盘尺寸草案

```
        ┌───────────────────┐
        │   操作面板/屏幕    │  ← 高度约 1.0m
        ├───────────────────┤
        │   ┌───────────┐   │
        │   │  清水箱    │   │  ← 中部
        │   │  15L      │   │
        │   ├───────────┤   │
        │   │  污水箱    │   │
        │   │  15L      │   │
        │   └───────────┘   │
        │   机械臂折叠收纳   │  ← 侧面/背部
        ├───────────────────┤
        │  电池 + 控制单元   │  ← 底部
        │  ┌──┐  刷盘  ┌──┐ │
        │  │轮│  吸水  │轮│ │
        └──┴──┴────────┴──┴─┘
         ←── 0.50m ──→
         深度约 0.65m
```

| 参数 | 值 | 说明 |
|------|------|------|
| 长 × 宽 × 高 | 650mm × 500mm × 1000mm | 通过 600mm 门宽需留 50mm 余量 |
| 清水箱 | 15L | 上部放置，重心偏高需配重优化 |
| 污水箱 | 15L | 紧邻清水箱下方 |
| 电池仓 | 底部居中 | 降低重心 |
| 清洁模组 | 底部前方 | 可更换式设计 |
| 机械臂 | 侧面折叠 | 作业时展开，行进时收纳 |

| 组件 | 规格 | 说明 |
|------|------|------|

## 6. 充电与能源系统

| 参数 | 规格 | 说明 |
|------|------|------|
| 电池类型 | 磷酸铁锂（LiFePO₄） | 安全性高、循环寿命长（≥ 2000 次） |

### 7.1 状态指示灯定义

| 灯效 | 颜色 | 含义 |
|------|------|------|
| 呼吸闪烁 | 蓝色 | 待机/空闲 |
| 常亮流转 | 绿色 | 正在清洁作业 |
| 缓慢闪烁 | 黄色 | 需要维护（缺水/满污/工具磨损） |
| 快速闪烁 | 红色 | 故障/急停 |
| 彩虹流转 | 多色 | 清洁完成展示（可选，提升品牌感） |
