## 1. 项目背景

公共卫生间是城市公共基础设施的重要组成部分，广泛存在于商场、地铁站、火车站、机场、公园、写字楼等场所。其清洁管理直接关系到市民出行体验、城市形象和公共卫生安全。

当前痛点：

| 痛点类别 | 具体表现 |
|---------|---------|
| 人力依赖 | 保洁人员老龄化严重，招工难、流动性大 |
| 管理粗放 | 依赖固定排班，无法根据实际使用量动态调度 |
| 响应滞后 | 设备故障、耗材缺失需人工巡查才能发现 |
| 质量难控 | 清洁标准主观性强，缺乏量化验收机制 |
| 数据缺失 | 无法沉淀运营数据，难以做成本优化和趋势分析 |
| 体验薄弱 | 使用者投诉渠道单一，缺少实时互动反馈 |
| 卫生风险 | 人工清洁存在交叉感染风险，疫情期间尤为突出 |

## 2. 项目愿景

PRISM 的终极目标是研发一台能够**全自动完成公共卫生间清洁作业**的智能机器人，同时配套完善的**智慧管理平台**形成完整的闭环运营体系。

项目分为两大维度：

| 维度 | 定位 | 文档覆盖 |
|------|------|---------|
| **平台维度**（01-05） | 感知、调度、质检、公众互动、数据分析的云端管理平台 | 已完成 v0.1 |
| **机器人维度**（06-10） | 机器人本体的需求、形态、功能、技术栈、研发规划 | 本次新增 |

两个维度最终融合：**机器人是"执行终端"，平台是"调度大脑"**。

## 3. 项目目标

### 3.1 机器人本体目标

1. **自主导航** — 在公共卫生间复杂环境中自主移动，适应湿滑、狭窄、障碍物密集的场景
2. **多模态清洁** — 具备地面冲洗/刷洗、便器清洁、台面擦拭、消毒喷洒等多种清洁能力
3. **智能感知** — 识别污渍类型与程度，自主规划清洁策略
4. **安全共存** — 在有人环境下安全运行，检测到人员自动避让或暂停
5. **自维护** — 自动回充、清/污水箱自动接驳、清洁工具自清洗

### 3.2 管理平台目标

1. **实时感知** — 通过 IoT 传感器采集人流量、环境指标、耗材余量等数据
2. **智能调度** — 基于数据驱动的动态清洁排班与任务派发（人机协同或全自动）
3. **质量闭环** — 清洁前后对比、AI 质检、主管复核的三级质量管控
4. **公众参与** — 扫码评价 / 一键报修 / 满意度调研
5. **数据决策** — 多维度看板、成本分析、趋势预测
